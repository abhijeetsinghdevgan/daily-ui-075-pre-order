# Challenge #075: Pre-Order Design - Nebula-1 VR

This project is a high-end pre-order landing page for a futuristic VR headset called **Nebula-1**. The design focuses on a premium, immersive aesthetic using dark mode, glassmorphism, and vibrant neon accents.

🔗 **Live Preview:** [https://abhijeetsinghdevgan.github.io/daily-ui-075-pre-order/](https://abhijeetsinghdevgan.github.io/daily-ui-075-pre-order/)

## 🚀 Key Features

- **Futuristic Aesthetics**: A cinematic dark theme with radial glows, glassmorphism, and custom typography (Orbitron & Inter).
- **Interactive Countdown**: A live JavaScript-powered timer to create urgency for the pre-order window.
- **Responsive Layout**: Optimized for all devices, from ultra-wide monitors to mobile phones.
- **Scroll Animations**: Subtle entry animations for features using the Intersection Observer API.
- **SEO Optimized**: Semantic HTML5 structure with proper meta tags and descriptive headings.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Custom design system, flexbox/grid, and keyframe animations.
- **Vanilla JavaScript**: Countdown logic and scroll interactions.
- **Google Fonts**: Orbitron (Headings) and Inter (Body).

## 📸 Design Preview

The hero section features a high-fidelity image of the Nebula-1 headset, surrounded by glowing elements and a clear Call to Action (CTA) for pre-ordering.
